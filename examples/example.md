# Tears of the Kingdom - Example Markdown

## Features

This document demonstrates how the **Tears of the Kingdom** theme handles Markdown formatting.

### Heading Levels

Headers are styled with the type/class color (#6E8FA6) to make them stand out.

### Text Formatting

- **Bold text** appears in keyword color
- *Italic text* also uses keyword color
- `Inline code` is highlighted in function color
- [Links](https://github.com) use the primary accent color

### Code Blocks

```typescript
// Code blocks with syntax highlighting
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

```python
# Python example
def calculate_damage(base_damage, multiplier=1.5):
    """Calculate damage with multiplier"""
    return base_damage * multiplier
```
