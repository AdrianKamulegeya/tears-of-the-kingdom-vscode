# Tears of the Kingdom VS Code Theme

A minimal, material-inspired VS Code dark theme based on The Legend of Zelda: Tears of the Kingdom.

## Color Palette

This theme features a carefully crafted color palette inspired by the game:

- **#0F1412** - Editor Background
- **#151C19** - Sidebar & Secondary Background
- **#1C2622** - Surface & Popups
- **#2A3A34** - Borders & Dividers
- **#C9D6CF** - Primary Text
- **#8FA39B** - Secondary Text & Comments
- **#5E746B** - Disabled & Subtle Text
- **#3FBF9A** - Primary Accent
- **#4FA6A3** - Functions
- **#C2A86B** - Keywords & Constants
- **#6E8FA6** - Types & Classes
- **#C06C5F** - Errors & Warnings
- **#223B34** - Selection Background
- **#1A2F28** - Active Line

## Installation

### From Source

1. Clone this repository
2. Copy the folder to your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions`
   - **macOS/Linux**: `~/.vscode/extensions`
3. Reload VS Code
4. Go to **Preferences > Color Theme** and select **Tears of the Kingdom**

### Manual Installation

1. Download or clone this repository
2. Open VS Code
3. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS)
4. Type "Install from VSIX" and select it
5. Navigate to the downloaded folder and select the extension

## Usage

1. Open VS Code
2. Go to **File > Preferences > Color Theme** (or press `Ctrl+K Ctrl+T`)
3. Select **Tears of the Kingdom** from the list
