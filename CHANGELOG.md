# Change Log

All notable changes to the "Tears of the Kingdom Theme" extension will be documented in this file.

## [0.1.0] - 2026-01-20

### Added
- Initial release of Tears of the Kingdom theme
- Dark theme with material-inspired design
- Complete color palette based on The Legend of Zelda: Tears of the Kingdom
- Syntax highlighting for various programming languages
- Consistent UI colors across all VS Code elements
- Support for terminal colors
- Git decoration colors
- Error, warning, and info indicators
